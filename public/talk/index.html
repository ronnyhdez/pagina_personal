Crear páginas web con Rmarkdown, MkDocs y GitHub pages una alternativa para documentación.

Una solución para evitar perder tiempo buscando lo que ya hicimos o evitar volver a navegar en el mar de información que hay en internet, es mantener notas en un lugar centralizado, fácil de acceder y con herramientas de búsqueda. Mostramos cómo usar el paquete docmaker para automatizar todos los pasos para pasar de tener archivos .Rmd a MkDocs y desplegarlos como página web usando GitHub pages.
